・ワイルド古里（黒色　合計8枚）：カードを出すと、次の人が出すカードの色を指定できる。次の人は色があっていれば何を出しても良い
・JP（各色一枚　合計5枚）：次の人はカードを2枚引く、という効果
・栗原さん（各色一枚　合計5枚）：順番逆周り　という効果

基本ルールはだいたいウノ
・自分の手番に出して良いカードを出す
・出せない場合は１枚引く（その時引いたカードが出せるなら出しても良い）
・最初に手札を0枚にした人が勝ち
・基本的には、同じ色のカードを場に出せる
・クレドの最後のワードが出た後は別のクレドの最初のワードを出せる（別の色でも出せる）
・手札が残り1枚になったら「フリート！！（うちの会社名）」と叫ぶ
・言ってないことを他の人に指摘された瞬間、2枚引く（指摘されなければ黙って上がれる）
・クレド上、文章として次に繋がるフレーズは同じに出すことができる
OK例）「cool」+「and」 とか「are our」+「ultimate goals.」とかも可能
NG例）「and」+「stylish」+「常に」みたいな日本語英語混合は不可　
・特殊カード上がり無し
・間違えたカードを出した（ことを他人に指摘された時）、1枚引く
・特殊カードについて：前に出されたカードと同じ色のカードしか出せない。これらの特殊カードの次は同じ色のカードを出せる（文頭に限らない）
